## 本程序的前缀：input代表需要用户输入，val代表计算后返回的数据，State代表中间过程

##？代表可以进行选择，！代表仅在部分条件下可用

# 程序主要由状态、输入和输出共三部分依次组成

0. 导入CoolProp数据库

import CoolProp
import CoolProp.CoolProp as CP

1. 用户选择状态方程（默认为HEOS，即多参数Helmholtz）

## 状态
# 状态.方程
input_EOS = 'HEOS' # 可为HEOS、PR、SRK

2. 用户选择纯质或二元混合物（默认为纯质）

2.1 选择纯质时，输入一个物质

# ？默认为纯质，可更改为混合物（仅在状态.方程为PR或SRK时可用）
# 状态.纯质
input_System = 'CO2' #详细信息及物质列表位于Data_substance.xlsx

2.2 当1处状态方程为PR或SRK时，可选择二元混合物，此时需要输入两种物质、组分和交互系数kij

# 状态.混合物
input_System = 'Heptane&Octane' #混合物
input_comp1 = 0.9
input_comp2 = 1-input_comp1 # 不用输入，由1-组分1获得
input_kij = -0.05

3. 后台应用状态（通过以下代码自动进行）

3.1 纯质

# 状态.纯质.应用
State_input = CoolProp.AbstractState(input_EOS,input_System)

3.2 二元混合物

State_input = CoolProp.AbstractState(input_EOS,input_System)
State_input.set_mole_fractions([input_comp1,input_comp2])
State_input.set_binary_interaction_double(0,1,"kij",input_kij)

4. 用户输入物质状态（默认为任意状态，input_Q为0）

## 输入
input_Q = 0 # 0或1（0为液态，1为气态）

5. 用户选择物性（默认勾选ρ、h、s、cp、cv和c），根据选择的物性，显示对应的表格。前两列固定为温度和压力，第三列开始为ρ（ρ和D表示的都是密度）等。

5.1 仅在状态.方程为HEOS，状态.纯质，状态为任意状态时可选

5.2 仅在状态.方程为HEOS，状态.纯质，状态为饱和液或饱和气时可选

7. 用户在表格中输入数据

7.1 当4处的状态为饱和气或饱和液时，用户只能输入T或P

input_T = 300 # 温度，K
#或者
input_P = 2*1E6 # 压力，单位为Pa，实际输入一般使用单位MPa，因此要在输入的基础上乘1E6

7.2 当4处的状态为任意状态时，用户可以输入T、P、D、h和s中的任意2个

# 以下任选2个
input_T = 300 # 温度，K
input_D = 600 # 密度，kg/m3
input_h = 280000 # 焓，J/Kg
input_s = 1263 # 熵，J/Kg-K

8. 后台应用状态

8.1 当1为HEOS，4处的状态为任意状态时，根据7.2用户的选择六选一。比如用户输入T和D，那就选第2个。

# ？默认为双输入（一般状态），可更改为单输入（饱和状态），根据输入变量从以下选择对应的一个
# 输入.双输入应用
#！对于状态.混合物仅可使用PT_INPUTS，状态.方程为PR、SRK时仅可使用PT_INPUTS或DmassT_INPUTS
State_input.update(CoolProp.PT_INPUTS, input_P, input_T)
State_input.update(CoolProp.DmassT_INPUTS, input_D, input_T)
State_input.update(CoolProp.DmassP_INPUTS, input_D, input_P)
State_input.update(CoolProp.HmassP_INPUTS, input_h, input_P)
State_input.update(CoolProp.PSmass_INPUTS, input_P, input_s)
State_input.update(CoolProp.SmassT_INPUTS, input_s, input_T)

8.2 当1为PR或SRK，2为纯质，4处的状态为任意状态时，根据7.2用户的选择二选一。比如用户输入T和P，那就选第1个

State_input.update(CoolProp.PT_INPUTS, input_P, input_T)
State_input.update(CoolProp.DmassT_INPUTS, input_D, input_T)

8.3 当1为PR或SRK，2为二元混合物，4处的状态为任意状态时，用户只能选择输入T和P

State_input.update(CoolProp.PT_INPUTS, input_P, input_T)

8.4 当1为HEOS、PR或SRK，4处的状态为饱和气或饱和液时，根据7.1用户的选择二选一。比如用户输入T，那就选第1个。

# 输入.单输入应用（二选一）
State_input.update(CoolProp.QT_INPUTS, input_Q, input_T) # 输入T
State_input.update(CoolProp.PQ_INPUTS, input_P, input_Q) # 输入P

9. 内部计算基本性质

9.1 纯质，共4个量

## 输出
# 应用.基本性质（纯质）
val_Tc = CP.PropsSI("Tcrit",input_System) # 临界温度，K
val_Pc = CP.PropsSI("Pcrit",input_System) # 临界压力，Pa
val_rhoc = CP.PropsSI("rhocrit",input_System) # 临界密度，kg/m3
val_molar = State_input.molar_mass() # 摩尔质量，kg/mol

9.2 二元混合物。共5个量

# 应用.临界性质（混合物）
mat_cri = State_input.all_critical_points()
for pt in mat_cri:
    val_Tc = pt.T # 临界温度，K
    val_Pc = pt.p # 临界压力，Pa
    val_rhoc = pt.rhomolar*State_input.molar_mass() # 临界密度，kg/m3
    val_molar = State_input.molar_mass() # 摩尔质量，kg/mol
    val_stable = pt.stable # 状态是否稳定

10. 内部计算一般性质

val_p = State_input.p() # 压力，Pa
val_rhomass = State_input.rhomass() # 密度，kg/m3
val_cp = State_input.cpmass() # 比定压热容，J/kg-K
val_cv = State_input.cvmass() # 比定容热容，J/kg-K
val_u = State_input.speed_sound() # 音速，m/s
val_h = State_input.hmass() # 焓，J/Kg
val_s = State_input.smass() # 熵，J/Kg-K
val_kT = State_input.isothermal_compressibility() # 等温压缩系数，1/Pa
val_ap = State_input.isobaric_expansion_coefficient() # 等压膨胀系数，1/K
val_kS = 1/val_rhomass/val_u**2 # 绝热膨胀系数，1/Pa
# 仅在状态.方程为HEOS，状态.纯质时可用
val_vis = State_input.viscosity()# 粘度，Pa-s
val_thercom = State_input.conductivity() # 导热系数，W/m-K
#！仅在状态.方程为HEOS，输入.单输入应用时可用
val_surten = State_input.surface_tension() # 表面张力，mN/m

11. 用户点击计算后在表格中显示结果
